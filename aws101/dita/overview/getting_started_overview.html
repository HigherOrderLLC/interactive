Getting Started: Putting It All Together Overview

After completing this module, you will:
  • have created an AWS account.
  • have had hands-on practice with Amazon Route53.
  • have launched an Amazon EC2 instance.
  • have become familiar with and used Elastic Load Balancing
  • have created an Auto Scaling group.
  • have created an Amazon RDS database.
  • have been introduced to DynamoDB.
  • have created an S3 bucket and put objects into it.
  • have created a download distribution with Amazon CloudFormation.
  • have created a streaming distribution and delivered videos with Amazon CloudFormation.
  • have recreated the Web Hosting Architecture diagram.
