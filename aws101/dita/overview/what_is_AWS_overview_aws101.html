Introduction to AWS Overview

In this course, you will:
  • Associate a traditionally virtualized environment with the AWS hosted computing model.
  • Be introduced to AWS products.
  • See AWS products working together by interacting with a Web Hosting architectural diagram.
  • Be able to articulate Distributed Computing Concepts such as Designing for Failure and Elasticity.
  • Take inventory of AWS Storage and Compute options, and decide how to use them.
  • Differentiate between security responsibiliies through the Shared Security Model.
  • Complete Getting Started Exercises to gain hands on experience with AWS products and become familiar with the AWS Management Console.
  • Rebuild a web hosting architecture diagram to check your understanding.
  • Navigate to further resources to expand your knowledge.
